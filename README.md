# SSSTA Blog RSS collect

这个项目收集SSSTA各位强者博客的RSS订阅地址，以供后续的使用。

请有意被收录的强者为博客添加RSS支持并修改相应的文件。

下载xml文件并导入到rss阅读器以订阅。对于`inoreader`用户，可以直接使用[组合包](https://www.inoreader.com/bundle/0014cd63d611)（不确保实时更新）
